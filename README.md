# POO-Java-II

## LIFO vS FIFO

* LIFO é o principio onde o último a ser adicionado deve ser removido tipo de um topo de dados, exemplo simples é os de paletes em galpões de armazenamentos, onde o último adicionado é o primeiro a ser retirado já que não tem como retirar o palete que fica embaixo dos outros paletes.

* FIFO é o principio onde o primeiro a ser adicionado dever ser o primeiro a ser removido, exemplo é como chegar numa fila em qualquer estabelecimento comercial, onde o primeiro a chegar será o primeiro a ser atendido.

A diferença entre eles é a forma como são removido e a ordem que os elementos ou objetos são removidos.

## Aplicações Prática

* Jogos e processamento de imagens: usam Arrays Estáticos para armazenar mapas e mundo em jogos e para representar os pixels de uma imagem
* Playslist de música, navegação nas páginas Web e gerenciamento de memória: usam as listas Encadeadas, como os botões de avançar ou voltar na Web, e os botões de próxima e anterior nas playlist de músicas
* Verificação de sintaxe em compiladores, operações aritméticas em calculadoras.
* Contadores de tíquetes e gerenciamento de filas por senhas.
* Fila de impressão ou acesso a disco.
